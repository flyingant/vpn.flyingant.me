# vpn.flyngant.me

## Getting Started

The project is based on `webpack` and `sass`

### Prerequisites

```
yarn or npm
```

### Installing

1. `yarn install` or `npm install`
2. run dev `npm run dev`
3. build the final page `npm run build`
